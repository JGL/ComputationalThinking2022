Signing up to GitHub
GitHub is a website that enables you to use Git to store, edit and share all your code with other developers around the world. Git is software for tracking changes to and sharing those changes with other programmers.Git is open source, while GitHub is owned by Microsoft. Both are free to use. Github is the industry standard website for programmers - employers may look at your GitHub to see what kind of code you write and how you help others.
Sign up to GitHub here: https://github.com/signup - please use a non university email address to do this, so you can keep logging in after you've graduated. Pick a username that you wouldn't be embarassed to apply for a job with! I'll give you 5 minutes to do this.
Learning Git
I can't give you a complete tutorial on Git right now. GitHub provides a good set of learning resources here: https://docs.github.com/en/get-started/quickstart/git-and-github-learning-resources . I recommend going through this after the Intensive is over, you won't need it for the rest of the week.
Using editor.p5js.org with your GitHub login
The easiest way of coding with p5.js is to use the amazing online editor that p5.js itself provides: https://editor.p5js.org. It's totally free and you can log into it with you GitHub login.
While you try logging in, watch me complete the editor.p5js.org part of the p5.js get started tutorial.
Give it a try yourself! You have 10 minutes.
Downloading VSCode
Visual Studio Code or VSCode is a free and open source text editor that is designed for programming software. You can download it for free here: . The brilliant Daniel Shiffman has several different videos on how he codes locally (not using the p5.js online editor, only working on his own computer). It's the piece of software that I use to write code, write lectures and organise my todo lists.
Using VSCode with p5.js on your own computer
While you are downloading VSCode, I'll complete the other half of the p5.js get started tutorial, BTW we are using VSCode instead of the Sublime Text editor, which is very similar.
Give it a try yourself! You have 15 minutes.
Running a local server via the VSCode "Live Server" extension
Sometimes you'll need to run a local web server in order for your p5.js sketches to run on your local laptop. The p5.js get started tutorial has instructions for installing a server, but I prefer to use the VSCode Live Server extension to start up and shut down a server really easily. Follow along and install it for yourself while I demonstrate it. You have 5 minutes.
Finally, I'd like you to pick a p5.js example and first try it on the p5.js editor (don't forget to check the File > Examples menu!), and then get it running on your local machine - I'll show you how to do that myself first, demonstrating errors, the console and comments too. A few "Coding Train" links there, if you want to go into more detail!
