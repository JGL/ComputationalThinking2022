This afternoon we are going to complete four exercises from Code as a Creative Medium, which was written by Tega Brain and Golan Levin. As I said this morning, I'd really recommend picking up a copy if you can, it's full of brilliant reference, exercises and project briefs.
We are going to pick up the pace this afternoon, with 20 minutes for each of the four exercises.
A reminder from this morning: I'd recommend making these exercises on the online p5.js edtior at https://editor.p5js.org - easier for sharing with others and a nice way of building up a portfolio of p5.js sketches.
Let's discuss "Left or Right" before we all try it on our own on the p5.js editor. Any suggestions?
Next, let's talk about the "Billiard Ball" exercise, before we try it individually in the p5.js editor. Any thoughts?
Next is the "Seven Circles" exercise. How you make something that repeats a shape across a screen?
Finally, let's look at the "Transitioning Rectangles" exercise. You'll need to change several graphical elements for each repeated rectangle for this exercise.
